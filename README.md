# SYSMATE AR

SYSMATE 제품 GLB 모델을 웹에서 확인하는 AR/3D 뷰어입니다.

## 실행 주소

https://ailvers1.github.io/20260616/

## 지원 방식

- AR 지원 기기: 실제 공간의 바닥을 인식한 뒤 제품을 배치할 수 있습니다.
- AR 미지원 기기: 자동으로 3D 미리보기 모드가 열립니다.

## 권장 기기

- Android Chrome
- ARCore 지원 Android 기기
- HTTPS 환경

iPhone, PC, 일부 Android 브라우저에서는 WebXR AR이 제한될 수 있습니다. 이 경우 제품을 회전하고 확대해서 볼 수 있는 3D 미리보기로 전환됩니다.

## 포함 제품

- AI 키오스크 화이트
- 일반 스탠드
- 포토 키오스크 블랙
